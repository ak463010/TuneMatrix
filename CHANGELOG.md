# Changelog

All notable changes to TuneMatrix will be documented in this file.

This project follows [Semantic Versioning](https://semver.org/) once public releases begin.

## [Unreleased]

### Added

- Open-source project guidelines, community files, issue templates, and release documentation.

### Changed

- Repository documentation prepared for external contributors.

## [0.1.0] - Unreleased

### Added

- MVP PySide6 desktop application.
- Multi-song import with project save/load support.
- Duration, BPM, rough key, relative key, and compatible-key analysis.
- Native Essentia helper integration with `librosa` fallback.
- Demucs stem-separation workflow.
- Tempo matching and key-tonic matching.
- Processed-output export flow that does not modify original files.
- Local unittest suite and GitHub Actions CI.
